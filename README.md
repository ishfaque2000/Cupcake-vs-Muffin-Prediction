# 🧁 Muffin vs Cupcake — ML Classification

Binary classification on a recipe dataset using **Logistic Regression** and **KNN**.  
Built as part of my ML coursework at **Sukkur IBA University**.

---

## 📂 Files
| File | Description |
|---|---|
| `recipes_muffins_cupcakes.csv` | Dataset (20 recipes, 8 features) |
| `Cupcake_vs_Muffin_Prediction_SVM_KNN_Classification.ipynb` | Main notebook |

---

## 🔍 What's Inside

**EDA** — class balance, box plots, correlation heatmap, Sugar vs Butter scatter  
**Model 1** — SVM   
**Model 2** — KNN with K-selection graph (K=1 to 15) to find the optimal K  
**Comparison** — accuracy bar chart of both models side by side
